# Characterization-of-Artificial-Intelligence-Accelerators-for-Timing-Analysis-Appendix
Appendix for Characterization of Artificial Intelligence Accelerators for Timing Analysis Master thesis. 
### The files provided in this appendix are as follows:
1. The code used to execute the AI model is provided in runAIModel.py.
2. The metrics collected from Nvidia Nsight Compute is available in the *.ncu-rep files.
3. The Worst-case execution time analysis is carried out in the Analysis.xlsx Excel file. Each sheet in the file corresponds to one kernel analysis.
